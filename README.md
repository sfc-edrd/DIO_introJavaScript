# Curso Digital Innovation One Recriando a Interface do Netflix

**Interface desenvolvida durante o curso ministrado pelo Felipe Aguiar.**

Neste projeto foram utilizados conceitos de semântica em HTML e técnicas de responsividade em CSS, incrementando a experiência do usuário final. 

Aqui, utilizamos duas bibliotecas externas:

 - FontAwesome
> Disponível em https://kit.fontawesome.com, introduz ícones utilizando CSS.
- Owl Carousel
> Disponível em https://owlcarousel2.github.io/OwlCarousel2/, cria menu em estilo carrossel. Desenvolvido em jQuery.
